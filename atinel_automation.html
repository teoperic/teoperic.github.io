Siemens PLC Automation

Internship at ATINEL d.o.o. | Varaždin, Croatia

In this project, I programmed Siemens S7 series PLCs using the STEP7/TIA Portal environments to automate production processes. I focused on developing reliable and efficient control logic to manage industrial operations. A key component of the project involved writing and optimizing a sorting algorithm directly in STL (Statement List), a low-level assembly-like language used in Siemens environments. The STL implementation allowed for fast sorting algorithm critical for real-time operations of PLCs. This project strengthened my understanding of PLC architecture, cycle time optimization, and hardware-software integration in automation systems.
